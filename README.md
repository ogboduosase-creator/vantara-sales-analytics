# Vantara Consumer Goods — Sales Analytics Simulation

**Tools:** Microsoft Excel | Pivot Tables | Dashboard Design
**Industry:** FMCG / Consumer Goods
**Type:** End-to-end self-directed analytics simulation

## Project Overview
A fictional Nigerian FMCG company used to simulate a real analyst workflow — from raw messy data through to an interactive Excel dashboard. Built across four phases.

## Phases
- **Phase 1 — Data Cleaning:** Removed duplicates, fixed formats, handled missing values
- **Phase 2 — EDA:** Revenue by region, product, category, and time period
- **Phase 3 — Dashboard:** Interactive Excel dashboard with dynamic slicers
- **Phase 4 — Reporting:** Findings report with business recommendations

## Key Findings
- Total net revenue: ₦58,314,400
- Top region: Port Harcourt — ₦12.8M
- Top category: Personal Care — 44% of revenue
- Top product: Dove Body Lotion — ₦8M
